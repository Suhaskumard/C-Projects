# 📚 Library Management System (C - CLI)

## 📌 Description

A feature-rich command-line Library Management System written in C.
It allows users to efficiently manage books with multiple operations.

---

## 🚀 Features

* ➕ Add books
* 📖 View all books
* 🔍 Search books by title
* 📤 Issue books
* 📥 Return books
* ❌ Delete books
* 📊 Count total books

---

## 🛠 Requirements

* GCC Compiler (or any C compiler)

---

## ⚙️ Compilation

```bash
gcc library.c -o library
```

---

## ▶️ Run Program

```bash
./library
```

---

## 🧑‍💻 Usage

1. Run the program
2. Select an option from the menu
3. Enter required details

---

## ⚠️ Limitations

* No file storage (data lost after exit)
* Fixed array size (max 100 books)

---

## 🔮 Future Improvements

* 💾 File handling (persistent storage)
* 🧠 Use linked lists / dynamic memory
